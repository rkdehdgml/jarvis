--------------------------------------------------------------------------------
1. 소프트웨어 정의
--------------------------------------------------------------------------------

"JARVIS Personal AI Assistant" (이하 "소프트웨어")는 개인 생산성 향상을 위한
AI 비서 애플리케이션으로, 다음 구성 요소를 포함합니다:

  · Electron 기반 데스크탑 HUD 오버레이
  · Java Spring Boot 백엔드 서버 (H2 내장 데이터베이스 포함)
  · Python 음성/동작 인식 센서 모듈
  · AI 엔진 연동 모듈 (Ollama / Anthropic Claude / OpenAI)

--------------------------------------------------------------------------------
2. 라이선스 부여
--------------------------------------------------------------------------------

본 소프트웨어는 개인 사용 목적으로 제공됩니다. 사용자는 다음 권리를 갖습니다:

  · 개인 소유의 컴퓨터에 설치 및 실행
  · 개인 목적의 수정 및 커스터마이징
  · 비상업적 목적의 배포

--------------------------------------------------------------------------------
3. 제한 사항
--------------------------------------------------------------------------------

다음 행위는 명시적으로 금지됩니다:

  · 소프트웨어의 상업적 판매 또는 재배포
  · 소프트웨어를 이용한 영리 목적의 서비스 제공
  · 악의적 목적(개인정보 침해, 불법 자동화 등)을 위한 사용

--------------------------------------------------------------------------------
4. 타사 서비스 및 API
--------------------------------------------------------------------------------

본 소프트웨어는 선택적으로 다음 타사 서비스와 연동될 수 있습니다:

  · Anthropic Claude API  —  https://anthropic.com/terms
  · OpenAI API            —  https://openai.com/terms
  · ngrok                 —  https://ngrok.com/tos
  · Ollama (로컬 실행)    —  https://ollama.ai

각 서비스의 이용 약관은 해당 서비스의 정책을 따르며, 본 소프트웨어는 그에
대한 책임을 지지 않습니다. API 키 및 인증 토큰은 사용자 본인이 관리합니다.

--------------------------------------------------------------------------------
5. 데이터 및 개인정보
--------------------------------------------------------------------------------

  · 모든 대화 기록 및 설정은 사용자 로컬 기기에만 저장됩니다.
    (위치: %APPDATA%\jarvis-project)
  · 외부 AI API 사용 시, 해당 API 제공사의 개인정보 처리방침이 적용됩니다.
  · 소프트웨어는 사용자의 동의 없이 데이터를 외부로 전송하지 않습니다.

--------------------------------------------------------------------------------
6. 면책 조항
--------------------------------------------------------------------------------

본 소프트웨어는 "있는 그대로(AS IS)" 제공됩니다. 개발자는 다음 사항에 대해
명시적 또는 묵시적 보증을 하지 않습니다:

  · 특정 목적에의 적합성
  · 무중단 또는 오류 없는 동작
  · 소프트웨어 사용으로 인한 직·간접적 손해

OS 자동 제어 기능(마우스/키보드 자동화) 사용 시 발생하는 결과에 대한 책임은
전적으로 사용자에게 있습니다.

--------------------------------------------------------------------------------
8. 제거 및 데이터 삭제
--------------------------------------------------------------------------------

소프트웨어 제거(언인스톨) 시:

  · 프로그램 파일은 설치 디렉터리에서 완전히 삭제됩니다.
  · 사용자 데이터(%APPDATA%\jarvis-project)는 삭제 여부를 별도로 확인합니다.
  · Windows 시작 프로그램 등록이 자동으로 해제됩니다.
